- Каталог сектора:

handlers.py - модуль со всеми хендлерами сектора
states.py - модуль с классом со всеми состояними FSM для этого сектора
create_database.py - модуль с функцией создания таблицы базы данных для этого сектора(в нем так же создается каталог 'database' в каталоге этого сектора в котором и будут храниться файлы баз данных сектора)

modules - каталог со всеми частными модулями сектора
database - опциальный каталог создающийся в create_database.py

- Список общих имен для частных модулей секторов:

message.py - модуль со словарем со всеми текстами для сектора
database.py - модуль с функциями для работы с базами данных сектора
